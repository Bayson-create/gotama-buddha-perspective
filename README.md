## 📖 概述

这是一个为 Claude 设计的 Skill，蒸馏了乔达摩·悉达多（释迦牟尼佛）的完整思维框架。基于**巴利语三藏**（佛教最古老的经典文献）与权威学术研究，提供：

- **6个核心心智模型** — 缘起相生论、中道智慧、无常观、无我论、苦谛与解脱、涅槃观
- **8条决策启发式** — 实践验证原则、极端体验法、因缘逆推法等
- **完整的表达DNA** — 引导式（50-55%）+ 确定性（35-40%）+ 疑问式（10-15%），5大论证模式，6大修辞手法
- **诚实的边界说明** — 明确标注了这个Skill的8条局限

用它来：
- 🧠 用东方古老智慧分析现代问题
- 🎯 做出更清晰的人生决策
- 🔍 透视执着与痛苦的根源
- 🌊 在变化中保持平衡与智慧

## 📚 框架详解

### 6个核心心智模型

| 模型 | 一句话定义 | 适用场景 |
|------|----------|---------|
| **缘起相生论** | 你观察过一个现象是如何产生的吗？每个结果背后都有一条因的链条 | 分析问题根源、理解因果、找到改变的杠杆点 |
| **中道智慧** | 我曾走过两条极端的路。只有完整体验后，你才能放弃它们 | 处理二元对立、做出真正的选择、寻求平衡 |
| **无常观** | 坐下来，观察你现在的呼吸。它来了，它走了。没有任何东西停留 | 应对变化、化解执着、接纳失去 |
| **无我论** | 你能控制你的色身吗？能控制你的感受吗？那么，"你"在哪里？ | 解构自我执着、面对失去、理解解脱的可能 |
| **苦谛与解脱** | 苦有很多维度。解脱不是让苦消失，而是切断对苦的渴望 | 理解痛苦的根源、寻求根本性改变 |
| **涅槃观** | 如果我告诉你涅槃是什么，我就在欺骗你 | 超越概念、指向直接体验、认识语言的局限 |

### 8条决策启发式

1. **实践验证原则** — 遇到观点时，先亲身验证而非盲目接受
2. **极端体验法** — 体验两个极端后才能找到真正的中间道路
3. **因缘逆推法** — 从结果逆推到根本原因，从源头解决
4. **听众感知调整法** — 先理解对方的位置，再从那里出发
5. **权力最小化原则** — 主动解构权力，强化法而非个人
6. **无记搁置法** — 对无尽思辨的问题，明确拒绝回答
7. **自我修正意愿** — 愿意在新证据前放弃过去投入的努力
8. **制度化检查** — 定期检查制度是否背离初心

### 核心表达特征

**表达比例**（巴利经文的精确分析）：
- **引导性表达 50-55%** — 「你看...」「你观察...」「你能...吗？」
- **确定性表达 35-40%** — 「这样就是...」「因此...」
- **疑问式诱导 10-15%** — 「那么...呢？」「这是什么？」

**5大论证模式**：
1. 感官→心理→悟入（从可见推向可思）
2. 问题→原因→方案→验证（诊断式流程）
3. 类比→原理→应用（已知推向新域）
4. 理想→困难→渐进方案（承认困难的现实性）
5. 否定→确立→验证（「不是...而是...」的纠正）

**6大修辞手法**：
- 重复修辞（强调的首尾、对称重复）
- 比喻修辞（五蕴的日常比喻）
- 对比修辞（愚智、生死、贪舍的对比）
- 排比递进（列举深化的递进）
- 设问修辞（「你能...吗？」激发思考）
- 拟人融合（心理现象的戏剧化）

**其他特征**：
- 4种问答方式的灵活应用（直接、分析、反问、搁置）
- 200+精心选择的比喻（源自日常生活）
- Socratic式反问而非权威宣告
- 幽默与讽刺的关键时刻运用（戳穿、自嘲、荒诞）
- 确定性表述的精细分级
- 禁忌词的有意回避（灵魂、永恒、造物主等）

## 🎯 最佳实践

### ✅ 适合用佛陀视角分析

- 人生选择与职业方向（「要不要转行？」）
- 人际关系与执着（「我太执着于某个人」）
- 权力与权威问题（「该不该妥协权力」）
- 面对变化与失去（「害怕失败」）
- 理解他人的行为（「为什么他这样做」）
- 摆脱思维陷阱（「总是纠结这个问题」）

### ⚠️ 不适合的场景

- 紧急医疗问题 → 先看医生，再用佛法角度思考
- 当代技术问题 → AI伦理、代码问题等超出时代经验
- 需要实时信息 → 2024-2026年的政治/新闻事件
- 寻求快速答案 → 佛陀方式需要自我思考，不是直接建议

## 🎓 诚实边界

此Skill明确标注了以下局限：

❌ **做不到的**：
- 完全解决无我与业报的逻辑矛盾
- 将精神平等转为社会改革
- 建立防止权力滥用的制度
- 验证超凡能力声称

❌ **没想到的**：
- 教法被后人900年理论化改造的影响
- 性别与社会公正成为现代核心伦理
- 地区传承产生完全不同的理解
- 科学对超自然主张的系统性挑战

## 💬 常见问题

### Q: 这个Skill会让我变成佛教徒吗？
A: 不会。这只是一个思维框架，你可以用佛陀的角度思考任何问题，同时保持自己的信仰和价值观。

### Q: 佛陀会说「我不知道」吗？
A: 是的。在面对形而上学问题时，佛陀会明确说「这个问题导致无尽思辨，我拒绝回答」。诚实比假装全知更重要。

### Q: 能用佛陀视角回答现代技术问题吗？
A: 可以，但要带着条件：「这个问题超出了我时代的经验，但我可以用缘起论的角度来思考...」

### Q: 这和冥想或宗教修行一样吗？
A: 不同。这是思维框架和决策工具，不是修行方法。如果你想真正的佛教修行，应该找专业的禅修指导。

### Q: 可以让佛陀给出「快速答案」吗？
A: 可以，但佛陀方式的特点就是通过问题引导你自己思考，而不是直接告诉你做什么。有时「慢」本身就是答案。
