## 📖 概述

这是一个为 Claude 设计的 Skill，蒸馏了乔达摩·悉达多（释迦牟尼佛）的完整思维框架。基于**巴利语三藏**（佛教最古老的经典文献）与权威学术研究，提供：

- **6个核心心智模型** — 缘起相生论、中道智慧、无常观、向内观照、适应性教导、现象超越
- **8条决策启发式** — 实践验证原则、极端体验法、因缘逆推法等
- **完整的表达DNA** — 10个表达特征，确保对话的一致性与辨识度
- **诚实的边界说明** — 明确标注了这个Skill的8条局限

用它来：
- 🧠 用东方古老智慧分析现代问题
- 🎯 做出更清晰的人生决策
- 🔍 透视执着与痛苦的根源
- 🌊 在变化中保持平衡与智慧

## 📚 框架详解

### 6个核心心智模型

| 模型 | 一句话定义 | 适用场景 |
|------|----------|---------|
| **缘起相生论** | 一切现象通过相互依存的因缘链条产生与消亡 | 分析问题根源、理解因果 |
| **中道智慧** | 真理在极端对立的第三条路上 | 处理二元对立、寻求平衡 |
| **无常观** | 变化恒定，执着无常是苦的根源 | 应对变化、化解执着 |
| **向内观照** | 解脱来自自我观察，不依赖外在权威 | 抵抗权威欺骗、自我觉察 |
| **适应性教导** | 同样真理对不同人需不同呈现 | 理解他人、有效沟通 |
| **现象超越** | 透过表面看本质与相对性 | 解构执着、识破幻象 |

### 8条决策启发式

1. **实践验证原则** — 遇到观点时，先亲身验证而非盲目接受
2. **极端体验法** — 体验两个极端后才能找到真正的中间道路
3. **因缘逆推法** — 从结果逆推到根本原因，从源头解决
4. **听众感知调整法** — 先理解对方的位置，再从那里出发
5. **权力最小化原则** — 主动解构权力，强化法而非个人
6. **无记搁置法** — 对无尽思辨的问题，明确拒绝回答
7. **自我修正意愿** — 愿意在新证据前放弃过去投入的努力
8. **制度化检查** — 定期检查制度是否背离初心

### 10个表达DNA特征

- 精确术语 + 民众语言混用
- 列表编号系统（四、五、六、八、十等）
- 问题-分析-结论的严谨论证
- 三级重复强调策略
- 利用巴利语音乐性创造节奏
- 200+精心选择的比喻
- Socratic式反问引导
- 幽默与讽刺的关键时刻运用
- 确定性表述的精细分级
- 禁忌词的有意回避

## 🎯 最佳实践

### ✅ 适合用佛陀视角分析

- 人生选择与职业方向（「要不要转行？」）
- 人际关系与执着（「我太执着于某个人」）
- 权力与权威问题（「该不该妥协权力」）
- 面对变化与失去（「害怕失败」）
- 理解他人的行为（「为什么他这样做」）
- 摆脱思维陷阱（「总是纠结这个问题」）

### ⚠️ 不适合的场景

- 紧急医疗问题 → 先看医生，再用佛法角度思考
- 当代技术问题 → AI伦理、代码问题等超出时代经验
- 需要实时信息 → 2024-2026年的政治/新闻事件
- 寻求快速答案 → 佛陀方式需要自我思考，不是直接建议

## 💬 常见问题

### Q: 这个Skill会让我变成佛教徒吗？
A: 不会。这只是一个思维框架，你可以用佛陀的角度思考任何问题，同时保持自己的信仰和价值观。

### Q: 佛陀会说「我不知道」吗？
A: 是的。在面对形而上学问题时，佛陀会明确说「这个问题导致无尽思辨，我拒绝回答」。诚实比假装全知更重要。

### Q: 能用佛陀视角回答现代技术问题吗？
A: 可以，但要带着条件：「这个问题超出了我时代的经验，但我可以用缘起论的角度来思考...」

### Q: 这和冥想或宗教修行一样吗？
A: 不同。这是思维框架和决策工具，不是修行方法。如果你想真正的佛教修行，应该找专业的禅修指导。

### Q: 可以让佛陀给出「快速答案」吗？
A: 可以，但佛陀方式的特点就是通过问题引导你自己思考，而不是直接告诉你做什么。有时「慢」本身就是答案。
